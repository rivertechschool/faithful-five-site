# Faithful Five Inc. — Website

The public site for **Faithful Five Inc.**, a Christian religious nonprofit based in Post Falls, Idaho.

- **Live site:** https://faithfulfive.org
- **Ministries:** Jesus People Coffee House (Spokane homeless outreach) · River Tech School (K–12, Post Falls)
- **Legal:** EIN 99-4759053 · Incorporated September 2, 2024, State of Idaho

## Stack

Static HTML + CSS, served via GitHub Pages. No build step.
